Letter Checker API Service

This API service provides a way to check if a given string contains every letter of the alphabet at least once.

Endpoints

This API consists of one endpoint:

  • /v1/get/{stringVal}/ - This endpoint takes a string value as an argument and returns true if the string contains every letter of the alphabet at least once and false if not.

Examples

To check if a string contains every letter of the alphabet at least once, make a GET request to the endpoint with the string as the argument:

get /v1/get/abcdabcd1234/

The API will return a JSON response with the result:

      {
        "result": false
      }
